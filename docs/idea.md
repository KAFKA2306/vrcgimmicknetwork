# VRCギミック技術情報プラットフォーム

## 1. プロジェクト概要

### 1.1 プロジェクト名
VRCギミック技術情報プラットフォーム

### 1.2 背景と目的
VRChatにおけるアバターおよびワールドのギミック開発は、Unity、UdonSharp、アニメーション、シェーダーなど多岐にわたる知識と技術を必要とします[6]。しかしながら、特に日本語で体系的にまとめられた情報は依然として不足しており、初心者から中級者が直面する「暗黙の了解」や特有の仕様の解決が困難な状況にあります。
本プロジェクトは、これらの課題を解決するため、VRChatのギミック開発に関する実践的な知識、Tips、トラブルシューティング情報、さらには衣装改変手順などを集約し、誰もが容易にアクセスし、また情報を投稿できる日本語の技術情報プラットフォームを構築することを目的とします。
これにより、開発者の学習コストを低減し、VRChatコミュニティ全体の技術水準の向上、ひいてはコンテンツの質と量の発展に貢献します。

### 1.3 スコープ
* VRChatのギミック開発（Udon, UdonSharp, Animator, シェーダー等）に関する技術情報の投稿、閲覧、検索機能の提供[6][8]。
* アバターギミック、ワールドギミック、衣装改変手順、最適化Tipsなど、関連する幅広い情報の集約[9][10]。
* Wix.comをCMSプラットフォームとして採用し、Velo by Wixを活用したカスタム機能の開発と実装[1][2][4]。

## 2. 対象ユーザー

* VRChatのワールド制作者およびアバター改変を行うユーザー全般[10][11]。
* ギミック開発に着手したばかりの初心者から、特定の問題解決や技術向上を目指す中級者[6]。
* 上級者に対しては、自身の知識や経験を共有し、コミュニティへ貢献する場、また情報の監修者としての役割を提供。

## 3. システム構成

### 3.1 インフラストラクチャ
* **プラットフォーム**: Wix.com（フルマネージドホスティング）
  * ストレージ: Wixの標準プランに含まれるストレージ容量
  * データベース: Wixコンテンツマネージャー（内蔵データベース）
* **ドメイン**: Wixの提供するサブドメイン（`ag1i5rer.wixsite.com`）または将来的に独自ドメインを連携[4]
* **SSL証明書**: Wixにより自動的に提供されるSSL
* **画像・動画ファイルの取り扱い**:
  * 画像: Wixのメディアギャラリー機能を利用。最適化のためサイズと解像度に注意。
  * 動画: YouTube, Vimeoなどの外部動画共有サービスの埋め込みを推奨。

### 3.2 ソフトウェア
* **CMS**: Wix.com (ビジュアルエディタとコードエディタを併用)
* **開発言語/環境**:
  * **Velo by Wix**: JavaScriptベースのコーディングプラットフォーム[2][4]
  * **Git連携**: GitHub (`KAFKA2306/vrcgimmicknetwork`)リポジトリとの連携によるバージョン管理[2][3]
* **主要機能実装方法**:
  * **コードスニペット表示**: HTMLおよびCSS、JavaScript（highlight.jsなどの外部ライブラリ統合も検討）
  * **レスポンシブデザイン**: Wixのレスポンシブエディタを活用[1][4]
  * **ローカル開発環境**: Wix CLI (`wix dev`)を使用したVSCodeでの開発[2][3]

## 4. データ構造・コンテンツ設計

### 4.1 コレクションとコンテンツタイプ
* **コレクション（データベース）**:
  * `GimmickInfo`: VRChatギミックに関する技術情報
  * `Announcements`: お知らせ、運営情報、コラム等
* **タクソノミー (分類)**:
  * **カテゴリ**: 階層構造を持つ大分類[6][8]
    * 例: Udonギミック, アバターギミック, ワールド最適化, トラブルシューティング, 衣装改変, シェーダー
  * **タグ**: 階層構造を持たない横断的なキーワード[9]
    * 例: ミラー, スイッチ, 同期, Modular Avatar, lilToon, VRM, Unity2022, VRCSDK3

### 4.2 カスタムフィールド (`GimmickInfo` コレクション用)
* **タイトル**: テキスト
* **本文**: リッチテキスト
* **アイキャッチ画像**: 画像
* **Unityバージョン**: テキスト/ドロップダウン（例: 2019.4.31f1, 2022.3.6f1, その他）
* **SDKバージョン**: テキスト（例: VRCSDK3-AVATAR-2023.XX.XX）[7]
* **対象**: タグ（ワールド, アバター, 両方）
* **難易度**: テキスト/ドロップダウン（初心者向け, 中級者向け, 上級者向け）
* **前提知識・依存アセット**: リッチテキスト
* **動作確認環境**: テキスト（例: Windows 11, Unity 2022.3.6f1, VRChat Build XXXX）
* **カテゴリ**: タグ（上記カテゴリから）
* **タグ**: タグ（上記タグから）
* **制作者名**: テキスト

## 5. 主要機能仕様

### 5.1 コンテンツ投稿・編集機能
* **投稿権限**: 初期は運営者のみがWixコンテンツマネージャーからギミック情報を投稿。将来的にはメンバー投稿機能の追加を検討。
* **エディタ**: Wixビジュアルエディタとリッチテキストエディタを使用。
* **コンテンツ入力**: タイトル、本文、コードスニペット（HTMLコンポーネントでシンタックスハイライト）、画像、動画埋め込み。
* **画像アップロード**: 解像度とファイルサイズに注意し、最適化設定を行う。
* **メタ情報入力**: カテゴリ、タグ、およびカスタムフィールド情報を必須または推奨として入力。

### 5.2 コンテンツ閲覧・検索機能
* **トップページ**: ギミック情報の新着順一覧（リピーターまたはギャラリーコンポーネントで表示、自動ページネーション）[1][4]。
* **カテゴリ・タグページ**: カテゴリ別、タグ別のギミック情報一覧（動的コンテンツページを活用）。
* **詳細ページ**:
  * 記事タイトル、投稿日、更新日、制作者名。
  * 本文（自動生成目次機能も検討）。
  * コードスニペット表示（ハイライト機能付き）。
  * カスタムフィールド情報（Unityバージョン、難易度など）を見やすく表示。
* **検索機能**:
  * Wixの高度な検索機能を実装（Veloでカスタマイズ）。
  * 検索対象: タイトル、本文、カテゴリ、タグなど。
* **関連記事表示**: 詳細ページ下部に同一カテゴリや関連タグを持つ記事をリピーターで表示。

### 5.3 ユーザーインタラクション機能
* **コメント機能**: Wixの標準コメント機能または外部サービス（Disqusなど）を統合。
* **記事評価機能**: 「役に立った」「分かりやすい」などの評価ボタンをVeloで実装（任意）。

### 5.4 パフォーマンスおよびセキュリティ対策
* **パフォーマンス最適化**: 画像の最適化、遅延読み込み、コードの最適化。
* **セキュリティ**: Wixの標準セキュリティ機能を活用し、必要に応じて追加の認証機能を実装。

## 6. 開発フロー

### 6.1 ローカル開発環境の設定
1. Wix CLIをインストール（`npm install -g @wix/cli`）[2]
2. GitHubリポジトリのクローン（`git clone git@github.com:KAFKA2306/vrcgimmicknetwork.git`）[2]
3. 依存関係のインストール（`npm install`）
4. ローカル開発サーバーの起動（`wix dev`）[3][4]

### 6.2 コンテンツ更新フロー
1. Wixコンテンツマネージャーでギミック情報を作成・編集
2. ビジュアルエディタでページレイアウトを調整
3. Veloコードエディタで機能をカスタマイズ
4. ローカルで変更をテスト
5. GitHubにコミット・プッシュして変更を保存[2]

### 6.3 サイト公開フロー
1. Wixエディタの「公開」ボタンでサイトを公開[1]
2. 定期的なコンテンツ更新とメンテナンス

## 7. UI/UX設計指針

* **テーマカラー**: VRChatのイメージカラー（ブルー系）や、落ち着いたダークトーンを基調とし、信頼感と先進性を表現[1][4]。
* **レスポンシブデザイン**: スマートフォン、タブレット、PCなど、あらゆるデバイスで快適に閲覧できるようレイアウトを最適化[1][4]。
* **ナビゲーション**: ヘッダーに「新着記事」「カテゴリ一覧」「タグ一覧」「検索」などの主要な導線を配置。
* **ギミック詳細ページのレイアウト**:
  * 記事タイトル、重要なメタ情報（Unityバージョン、難易度など）をページ上部に表示。
  * コードスニペットは背景色と等幅フォントで区別し、コピー機能も実装。
  * 画像は適切なサイズでレイアウトし、必要に応じてライトボックス表示も検討。
* **可読性**:
  * フォント: ゴシック体ベースの読みやすいフォントを選定。
  * フォントサイズ: 本文は16px以上を基本とし、読みやすさを重視。
  * コード表示: 等幅フォントとシンタックスハイライトで可読性を向上。

## 11. まとめ

VRChatのギミック開発に関する日本語の技術情報を集約・共有するプラットフォームを、Wix.comとVelo by Wixを活用して構築します。GitHub連携によるバージョン管理と、ローカル開発環境での効率的な開発を行い、高品質なコンテンツを提供します。本プラットフォームが、VRChatクリエイターにとって価値ある情報源となり、コミュニティ全体の技術水準向上に貢献することを目指します。
